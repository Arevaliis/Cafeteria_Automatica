# Cafetera Automática ☕

Este proyecto simula una cafetería automática en un instituto usando programación multihilo en Java. La máquina prepara cafés y varios profesores los consumen de forma concurrente, compartiendo un depósito limitado.

---

## 🖥️ Salida por consola

La salida del programa se obtuvo ejecutando la aplicación Java desde la terminal y redirigiendo la salida estándar a un archivo de texto usando:

````bash
java Main > ../../../resultados/resultado_11.txt
````
